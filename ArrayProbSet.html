Problem Set

  • "printReverse"

    prints the arrays in reverse

    [1,2,3,4] //4,3,2,1

    ["a","b","c"] //c,b,a

  • "isUniform"

    returns true if all elements are identical

    [1,1,1,1] //true

    [2,1,1,1] //false

    [a,b,p] //false

    [b,b,b] //true

  • "sumArray"

    returns the sum of all elements in array

    [1,2,3] //6

    [10,3,10,4] //27

    [-5,100] //95

  • "max"

    returns maximum number in each array

    [1,2,3] //3

    [10,3,10,4] //10

    [-5,100] //100
